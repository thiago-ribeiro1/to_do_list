# 📋 To-Do List

## ⚠️ Este é um projeto temporário desenvolvido com fins acadêmicos.

### 1. Rodar o Backend (Node.js + Express)

```bash
cd backend
npm install
node index.js
```

### 2. Rodar o Frontend (React + Vite)

```bash
cd frontend
npm install
npm run dev
```

O frontend será iniciado em: http://localhost:5173
